## 🌟 Project Overview

Real Estate CRM is a **full-stack web application** designed to manage properties, customers, and deals efficiently.
It provides a **role-based system** for Owners, Managers, and Customers with a clean and modern UI.

---

## 🎯 Key Features

✨ **Authentication System**

* Secure Login & Signup
* Role-based access control

🏠 **Property Management**

* Add / Edit / Delete properties
* View property listings

💼 **Deal Management**

* Track ongoing deals
* Manage sales pipeline

🔍 **Smart Search**

* Search properties, users, and deals instantly

📊 **Dashboard**

* Overview of business data
* Quick insights

## ⚙️ Setup & Installation

### 1️⃣ Clone the repo

```id="p4g8qk"
git clone https://github.com/shreya59195-bit/Real-Estate-CRM.git
cd Real-Estate-CRM
```

### 2️⃣ Install dependencies

#### ▶ Frontend

```id="z1b7yr"
cd frontend/frontend
npm install
npm run dev
```

#### ▶ Backend

```id="n2k4jd"
cd backend
npm install
npm start
```
